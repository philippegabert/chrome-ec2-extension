Version 5.7 - May 2021


What's new
  1. (v5.7) Chrome Tabs Permissions removed
    • Change in required permissions as per Google guidelines
  2. (v5.6) Payment method change
    • Change in payment methods as Google Payment has been deprecated.
    • Premium users' licenses details are now synced with the user sessions storage. Therefore if you switch computer, your premium order will automatically appear.
    • Bug fixes here and there
  3. (v5.5) Bug fixes
    • Bug fixes here and there
  4. (v5.4) Bug fixes
    • Spanish definition was not working anymore (Thanks Anna!).
  5. (v5.3) New option and improved spanish translation
    • A new option has been added to for the last search to be automatically displayed when reopening the extension.
    • Improved spanish version thanks to Alejandro
  6. (v5.2) Google Docs compatible
    • The extension now works on Google Docs !
  7. (v5.1) Catalan translation
    • The extension is now translated in Catalan. Thanks Javier :)
  8. (v5.0.8) Augmentation du nombre de recherches
    • Les utilisateurs Français peuvent augmenter le nombre de recherches en installant une extension supplémentaire.

      LeBonCoin Chrome Extension
  9. (v5.0.7) Pronunciation added
    • Pronunciation is added to the translation when available.
  10. (v5.0.4) "Command Key" trigger for Mac Users
    • So far, only the Ctrl Key was available to trigger "in-site" translation. It is now possible to trigger the in-site search with the "Command" key of Mac computers.
  11. (v5.0.2) "In-site" translation update
    • It's now possible to update the translation in the "in-site" search popup.
  12. (v5.0.2) Order ID
    • For premium users, the order ID is now visible in the option page.
  13. (v5.0.2) "In-site" default language selection
    • A message is now displayed when user select the "In-search" option without selecting a default language.
  14. (v5) In-site translation:

    When double clicking a word in a page, its translation can be automatically shown.
    • To activate it:
      • In the options page, select the following option:
      • To avoid trigger the translation everytime, you can select the option:

        When activated, translation will be enabled only when Ctrl key is pressed while double clicking.
  15. (v5) Automatic search:

    When selecting a word and opening the application, the translation will be automatically triggered. (No need to click on "Search" button or press enter)
  16. (v5) New translations !

    All the translations/dictionaries but english usages are now available in the application.
  17. (v5) Premium users

    With this new version comes the premium edition. All the "automatic" options are for premium users. Go premium is pretty simple and pretty cheap: Roughly $1,99 for the whole life. Good deal, right?
    Unlock all features !
    • No search limitation (Non premium users get a 10 searches limitation per day)
    • "In-site" search
    • Automatic search on text selection
    • Contextual menu (right click)
Bug fixes
  1. v5.0.9
    • Page encoding was not succesfully set, leading to strange asian characters for some users.
  2. v5.0.6
    • Last language used was not updated when using "in-site" search. Corrected.
  3. v5.0.5
    • Contextual menus were not showing up in some cases. Corrected.
  4. v5.0.3
    • One of the script messed up with the debugging console of Chrome. Corrected.
  5. v5.0.1
    • Some translations where not working properly when having accents in the words. (Ex: Spanish to English)
    • Positioning of popup page when "in-site" search could go out of screen when double clicking on edges of screen. The user had to drag the popup manually to bring it back into viewport.
    • CSS Injection has been reviewed to avoid messing with visited page.
    • Not really a bug fix, but... I've reduced the frequency of "Did you know" messages for non premium users.
  6. v5
    • Some translations were not working due to changes in WordReference pages
    • Contextual menus were not working
    • Some issues with the default translation have been fixed. Be sure to select the translations you want to see and your favorite one.
Deprecated
  1. Voice recognition inside extension has been removed. Chrome has removed voice recognition feature as there were some security issues with it.
